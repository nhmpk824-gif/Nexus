## 一句话

**关系是有形状的、被记住的、会回过头来找你的。**

## 你会注意到的变化

🧠 **她记得，并且会用** —— 显著性加权的记忆召回、dream cycle 生成的 1-3 条对你的反思、callback 队列（下次聊天她可能会温柔地提一句"上次你说的礼物挑了吗？"）、30 / 100 / 365 天的周年里程碑。

💝 **关系有了形状** —— 五级关系进度（陌生→相识→朋友→挚友→亲密）+ 四个子维度（trust / vulnerability / playfulness / intellectual）+ 心情感知召回（同一段记忆在不同心情下感觉会不一样）。

🤝 **关系有了类型**（v0.3.0 新加） —— Onboarding 和设置里新增"我们之间是哪种关系？"的选择：开放 / 朋友 / 导师 / 安静的陪伴。每种都用一行 prompt 偏置她的语气，但不会覆盖你的 SOUL.md。

💭 **"想着你"** —— 你长时间没说话时，她会按你设的关系类型推送一条系统通知。朋友模式："回来看看，我攒了点想聊的。" 安静陪伴模式："我在。回来时我都在。" 23-08 点静默不打扰。

🎬 **角落里的存在感** —— Autonomy V2 多了第 4 个动作 idle_motion（沉默的小手势：伸懒腰、歪头、打哈欠），不发文字，让她在你看屏幕的余光里"活着"。

🌅 **平滑的场景过渡** —— 白天到黄昏到深夜的背景图不再是某个时刻突然切换，而是 2 小时窗口内 smoothstep 缓动叠加。色彩对比也拉强了：黎明粉、金时刻深琥珀、深夜冷淡蓝。

🪟 **Liquid Glass UI** —— 紫色 accent 重塑（仅作为点缀，背景仍中性深色不晃眼）。窗口大小和位置现在跨启动持久化——拖到喜欢的尺寸下次就还是那样。

🌤️ **天气更精准** —— 小时级预报、体感温度 + 湿度、后天预报。不再是一句"今天有阵雨"。

## 接下来 v0.4 准备做什么

战略定位是"narrative composition"——不再造新的原语，而是把已有的组合成 ritual。

**Sprint 1（已开工）**：
- ✅ 关系类型声明 + 想着你通知（**这一版已 ship**）
- ⏭️ **早晚 bracket**（最高杠杆）：早 7-10 点抛一个轻问题，晚 21-24 点 callback。复用 callback 队列 + autonomy V2。"今天打算做点什么？" → "你早上说的那件事后来怎么样了？"
- ⏭️ **Sunday letter**：周日夜里 persona 写一封信总结这一周对话，可导出文本 / PDF。把"周"变成 keepsake。

**未上排的候选**（有人想要请回复）：
- Listen Mode —— 切到只听不说，AI 只表情 + 动作不发文字，事后写总结
- 多日 Arc —— AI 内部跑一个 5 天的小线索（"了解你的工作"），把问题 / callback 织进日常
- Future-self time capsule —— 给未来自己写信，scheduler 几个月后 persona 替你念
- Ambient mode —— 长时间无互动进入半透明只显示时间 + 偶尔内心独白的状态

## 反馈想听到的

特别想知道：

1. **callback 时刻** ("上次你说的礼物挑了吗") 触发频率合适吗？太多还是太少？
2. **idle motion** 看起来是"活着"还是"抽搐"？
3. 时间过渡的颜色对比——是不是真的看不到突兀的切换？
